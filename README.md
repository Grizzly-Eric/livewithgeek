## livewithgeek - 极客室友

这是一个用reactjs做的合租平台项目，后台用的是LeanCloud的数据库，直接ajax请求获取数据，操作数据。   
技术上准备加入reduce数据流模式，减少整页刷新操作。

#### 项目已包含功能：
+ 注册登录
+ 创建组队、加入组队
+ 队内留言交流

#### 项目还将完成的功能：
+ 房源录入
+ 组队分类筛选
+ 房源分类推荐


在线地址：[livewithgeek.com - 极客室友](http://geekwithreact.applinzi.com)
